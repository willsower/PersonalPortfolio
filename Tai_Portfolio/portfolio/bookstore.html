Bookstore Project

Individual Project

Dynamic Web Application that creates an Ecommerce website for ordering books. This website uses AJAX functionality to dynamically output titles searched. Uses cookies to store customer shopping items in cart for future checkout.

During customer checkout, will search for an existing email address in the database, and writes shipping information in form on the order confirmation page. The Customer ID is encryped using the Rijndael encryption algorithm. An email is sent to the customer with the order information.
