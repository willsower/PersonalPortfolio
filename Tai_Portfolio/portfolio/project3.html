Pre-Onboarding Skill - Alexa Hackathon

5 Person Team

The Pre-Onboarding Skill which was proposed by our team, was to solve the onboarding problem at Amazon. We proposed a skill that could be used by oncomming workers to ask when important documents are due, or whether they have been received on the other end. There was a need for this, since currently email was how oncomming workers were tracking their important paperwork.

This skill uses JavaScript as the primary language, we used DynamoDB to store user information. Lambda functions were used to access that information, and also update the information. This was intended to show, that when something gets submitted, the database would change accordingly, so users could get immediete results when they have completed something.
