Deadwood Board Game

2 Person Team

This game is a 2D replication of Deadwood by Cheapass games. My team member Daria and I started with the design phase, then created a text based implementation of the game, then finally added in the GUI using JavaFX.

For the design and organizational phase, we used UML Diagrams, Use Cases, and Sequence diagrams to organize our codebase. Then, we started implementing the text based version using Java, and created an XML parser to read XML files containing card information. Finally, we used JavaFX and CSS to create our final GUI product using the MVC framework, and other Object Oriented Programming concepts.

Video of Deadwood Gameplay

In this video, you will see 7 players. They will act, rehearse, move around the board, and upgrade. There is error checking, and the only actions that a player can perform will display. At the end of the game, a list of players who won will print out along with the overall score of each player. Ties are taken in account aswell.
