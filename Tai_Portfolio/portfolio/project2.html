Tex2Speech

5 Person Team

Designing and developing a text-to-speech application that can read LaTeX documents and give an audio response. For this project, it was brought to us by a client wanting to upload TeX documents and listening to them. Currently, there is no application out there that performs this operation.

This application is currently a work in progress. We are creating a parser that will pull LaTeX math mode, and latex commands/environments which we will feed to Amazon Polly (text-to-speech sythesizer) using a Speech Sythesis Markup Language (SSML). Using AWS tools, we will also host our web application through their services.
