Interactive Map Project

Individual Project

Interactive map to display salmon restoration projects throughout Whatcom County, Washington State. This map was contracted by the Nooksack Salmon Enhancement Association (NSEA). This map was added to NSEA's SquareSpace web application using JavaScript and MapBox.

There were two versions of this created. One being a React app which connected to MongoDB and pulled the map display from Mapbox. The other version is entirely using the MapBox API. In the end they both display the contents, allowing users to click on the icons which displays a tooltip of information.

Date

September 2020 - December 2020

Languages & Tools Used

HTML / CSS / JavaScript / React / MongoDB / MapBox / SquareSpace
