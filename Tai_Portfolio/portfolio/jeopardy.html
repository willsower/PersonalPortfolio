Jeopardy Magazine 57th Edition

Contracted Project

Web application to display the 57th edition's contents. Web developer and designer for Jeopardy magazine website. Worked with the literature team and the magazine design team in creating the 57th edition for Jeopardy submissions. Translated the physical magazine onto Jeopardy's website using SquareSpace and used front end languages to facilitate some interactions SquareSpace doesn't offer (e.g, JavaScript).

Date

January 2021 - June 2021

Languages & Tools Used

HTML / CSS / JavaScript / SquareSpace
