start

editing

To modify HTML files you should use text editor, we recommend VS Code or Sublime Text.

publishing

At first choose your homepage from HTML files and rename it to index.html, then upload files to your server (use FTP client, we recommend Filezilla).

Dark Theme

To enable dark theme add .dark-theme class to body

Lightbox

To open your media with Lightbox, create a link and add the attribute data-lightbox

Lightbox also supports Youtube, Vimeo and HTML5 videos


Links with the same data-lightbox attribute will open all of them as a collection.


Optionally you can add thumbnails by adding data-thumbnail attribute to links

Masonry

Use following attributes to setup Masonry Portfolio.

Attr Description
data-columns Column count
data-gutter Space between items

Portfolio Grid

Use following attributes to setup Portfolio grid.

Attr Description
data-columns Column count
data-gutter Space between items
data-cell-ratio Default item aspect ratio

To Change item sizes use .wide and .tall classes

Justified Grid

Attr Description
data-target-height Row height (target)
data-gutter Space between items

Portfolio Filters

For your portfolio items to assign category use classnames

Add filters with data-filter attribute and classname selectors

Typed Text

To add text typing animation use data-typed attribute with words (separated with semicolon).

Video Background

Use following attributes to setup video background. It supports Youtube, Vimeo and Local videos.

Attr Description
data-videobg-src Video URL (Youtube, Vimeo or Local video)
data-video-ratio (optional) default value is 16/9
data-poster Poster Image URL.

Thumbnail Videos

Thumbnail Video element will load original video (iframe) only after clicking play button. It supports Youtube, Vimeo and Local videos

Buttons

You can use 7 predefined styles for buttons

Appear Animations

Add appear animations with data-animation attribute. Set animation-name, duration and delay like in the example below.

You can use 4 predefined animation names fade-in-bottom fade-in-left fade-in-right fade-in-top.

Animate.css library is also supported.

Spacing Classes

Class Names like .px-5 .mb-0 .my-10 are Bootstrap Spacing Classes. Sizes are extended up to 15.

Backgrounds & Overlays

Set Backgrounds with data-background attribute. Set image URL, HEX or RGB color codes


Use data-overlay attribute to set overlay color and opacity (separated with semicolon)

Contact Form

Attr Description
data-mailto Recipients E-mail address
data-subject Subject Line

You can remove data-mailto and data-subject attributes from HTML and configure from php/mail.php file, LINE 3 and LINE 8
